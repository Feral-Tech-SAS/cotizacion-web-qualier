En esta versión revisada acordamos eliminar el CMS propio del alcance del proyecto, lo que permite reducir la inversión total en un 50% — de $3.900 a $1.950 USD. El sitio se desarrolla en tres fases independientes, siendo la Fase 1 el hub corporativo regional completo por $900 USD, con presencia unificada bajo el dominio qualierinternational.com. Es importante tener en cuenta que, al no incluir un panel de administración, cualquier actualización de contenido — textos, imágenes, datos por país u otros elementos — deberá realizarse directamente sobre el código fuente por parte de un desarrollador.

Stack Tecnológico Recomendado

Astro
Frontend · Sitio Público

Framework SEO-first. Renderizado híbrido SSG + SSR. Cero JS innecesario. Soporte nativo multiidioma ES/EN.

Supabase
Base de Datos · Formularios

PostgreSQL gestionado. Almacenamiento de envíos de formularios y datos de contacto. Preparado para API REST de Odoo en fases futuras.

Vercel
Hosting · Serverless

CDN global. Deploy automático desde Git. Funciones serverless para API y formularios. Soporte nativo Astro.

Contenido en Código
Sin CMS · Sin Licencias · Sin Dependencias

Textos, imágenes y datos se gestionan directamente en el repositorio Git. Máximo rendimiento y cero dependencias externas. Toda modificación requiere un desarrollador.

¿Por qué no WordPress u otras plataformas?  Un stack propio elimina licencias y dependencias, y permite integración directa con Odoo ERP vía API REST — algo que plataformas cerradas no facilitan sin plugins costosos o limitaciones técnicas.

Sobre «Presencia Regional» (Fase 1):  Esta sección presenta cada uno de los cuatro mercados del Grupo y enlaza de forma natural hacia el sitio, landing page o canal digital propio de cada país — ya sean plataformas actuales o recursos que se desarrollen en el futuro. El hub corporativo no duplica ni reemplaza los ecosistemas digitales locales de cada filial; los articula bajo una identidad regional unificada, amplificando la visibilidad de cada mercado desde un único punto de entrada.
