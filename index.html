Resumen Ejecutivo

Se propone el desarrollo de un sitio web corporativo regional para Qualier Internacional, con presencia unificada en cuatro mercados bajo el dominio qualierinternational.com. La solución contempla un CMS propio sin dependencia de plataformas externas, arquitectura preparada para integración futura con Odoo ERP, y un stack tecnológico moderno que garantiza rendimiento, escalabilidad y control total del código. El lanzamiento está proyectado para al finalizar el proyecto, alineado al rebranding oficial del Grupo.

Stack Tecnológico Recomendado

Astro
Frontend · Sitio Público

Framework SEO-first. Renderizado híbrido SSG + SSR. Cero JS innecesario. Soporte nativo multiidioma ES/EN.

Supabase
Base de Datos · Auth

PostgreSQL gestionado. Auth para el CMS. Almacenamiento de archivos. Preparado para API REST de Odoo.

Vercel
Hosting · Serverless

CDN global. Deploy automático desde Git. Funciones serverless para API y formularios. Soporte nativo Astro.

CMS Propio
Panel de Administración

En Astro, integrado al proyecto. Gestión de productos, blog, contactos y datos por país. Sin licencias.

¿Por qué no WordPress u otras plataformas?  Un stack propio elimina licencias y dependencias, y permite integración directa con Odoo ERP vía API REST — algo que plataformas cerradas no facilitan sin plugins costosos o limitaciones técnicas.

Integración con Redes Sociales — ¿Qué incluye?

Previsualizaciones Enriquecidas (OG)

Cuando cualquier URL del sitio se comparte en WhatsApp, LinkedIn, Facebook o Telegram, se genera automáticamente una tarjeta visual con imagen, título y descripción. Aumenta la tasa de clics en comunicaciones de marca.

Botones de Compartir y WhatsApp

Blog y fichas de producto con botones para compartir en redes. Cada sección de país incluirá un botón de WhatsApp Business con el número local de atención directa.

Píxeles de Seguimiento y Analytics

Facebook Pixel y LinkedIn Insight Tag para retargeting y audiencias similares. Google Analytics 4 para tráfico, conversiones y origen de visitas por país.

Links a Perfiles Oficiales

Cabecera y pie de página con íconos a Instagram, LinkedIn y Facebook. Configurables desde el CMS sin tocar código, diferenciables por país si se requiere.

SEO Social y Schema Markup

Metadatos estructurados Schema.org para empresa, productos, ubicaciones y artículos. Mejora el posicionamiento orgánico y las búsquedas de marca en Google.

Feed de Instagram (Opcional)

Últimas publicaciones del Instagram corporativo en la página de inicio. Actualización automática. Requiere cuenta de Instagram Business verificada.
